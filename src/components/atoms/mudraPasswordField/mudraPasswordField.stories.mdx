# Password Field

Password field can be use on login forms or any other form to keep password protected.

## Usage

### 1. Password Field

Normal password field where user can type password and can read it by clicking eye button.

export const PasswordField = () => {
  return (
    <MudraPasswordField
      disabled={false}
      label="Password"
      placeholder="Please type password"
      errorMessage=""
    />
  );
};

<Canvas>
  <Story name="PasswordField">{PasswordField.bind()}</Story>
</Canvas>

```jsx
import { MudraPasswordField } from "@services/storybook-storybook/atoms/index";

export const PasswordField = () => {
  return (
    <MudraPasswordField
      disabled={false}
      label="Password"
      placeholder="Please type password"
      errorMessage=""
    />
  );
};
```

### 2. Disabled Password Field

Password field is disabled where user neither can type anything nor can read.

export const DisabledField = () => {
  return (
    <MudraPasswordField
      disabled={true}
      label="Password"
      readOnly={false}
      placeholder="Please type password"
      defaultValue="John Doe"
    />
  );
};

### 3. ReadOnly Password Field

Password field is readOnly where user neither can type anything nor can read.

  return (
    <MudraPasswordField

      readOnly={true}

## Props

`MudraPasswordField`

<div style={{overflowX: 'auto'}}>

| prop               | description                                                     | type       |
| ------------------ | --------------------------------------------------------------- | ---------- |
| `label`            | label of password input field (Required)                        | `string`   |
| `disabled`         | if true, input field will be disabled. (optional)               | `boolean`  |
| `readOnly`         | if true user can read it.                                       | `boolean`  |
| `placeholder`      | placeholder of input field.                                     | `string`   |
| `defaultValue`     | default value of input field                                    | `string`   |
| `className`        | user can use this className for giving custom styles (optional) | `string`   |
| `errorMessage`     | error message on wrong password                                 | `string`   |
| `onChangeCallback` | callback function for change on input field                     | `function` |

</div>
