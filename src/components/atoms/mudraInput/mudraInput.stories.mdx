# Input

- Input Fields let users enter and edit text.
- Input Field comes with 6 different states `"default (default)"`, `"focused"`,` "filled"`, `"disabled`", `"error"`, `"read-only"`.
- Input Field comes with 8 different features `"normal (default)"`, `"left-icon`", `"right-icon"`, `"left-right-icon"`, `"indicative-text"`, `"indicative-text-left-icon"`, `"link"`, `"link-left-icon"`.

## Usage

## 1. Basic Input

The Input wrapper component is a complete form control including a label, input, placeholder and help text.

<Canvas>
  <Story
    name="Basic Input"
    args={{
      label: "Text - Normal",
      placeholder: "Input Text",
      state: "default",
      feature: "normal",
      hintText: "Hint Text",
      hint: false,
    }}
  >
    {InputTemplate.bind()}
  </Story>
</Canvas>

```jsx
import { MudraInput } from "./@services/storybook-storybook/atoms/mudraInput";

const Component = () => {
  return (
    <MudraInput
      label="Text - Normal"
      placeholder="Input Text"
      state="default"
      feature="normal"
      hintText="Hint Text"
    />
  );
};
export default Component;
```

## Props

<div style={{overflowX: 'auto'}}>

#### `MudraInput`

| prop         | description                                                                                                                                                                                                  | type                                |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------- |
| className    | Override or extend the styles applied to the component.                                                                                                                                                      | `string`                            |
| autoFocus    | If true, the input element is focused during the first mount.                                                                                                                                                | `boolean`                           |
| label        | A text to display the label.                                                                                                                                                                                 | `string`                            |
| value        | The value of the input element, required for a controlled component.                                                                                                                                         | `string`                            |
| placeholder  | The short hint displayed in the input before the user enters a value.                                                                                                                                        | `string`                            |
| defaultValue | The default value. Use when the component is not controlled.                                                                                                                                                 | `string`                            |
| prefixIcon   | Icon to be displayed on the start of the Input.                                                                                                                                                              | `SVG Icon`                          |
| suffixIcon   | Icon to be displayed on the end of the Input.                                                                                                                                                                | `SVG Icon`                          |
| hintText     | A text for displaying the hint the below the Input.                                                                                                                                                          | `string`                            |
| state        | Represents different state of the component `"default (default)"`, `"focused"`,` "filled"`, `"disabled`", `"error"`or `"read-only"`.                                                                         | `string`                            |
| disabled     | If true, the component is disabled.                                                                                                                                                                          | `boolean`                           |
| hint         | If true, the hink text will be displayed.                                                                                                                                                                    | `boolean`                           |
| feature      | Represents different features of the component `"normal (default)"`, `"left-icon"`, `"right-icon"`, `"left-right-icon"`, `"indicative-text"`, `"indicative-text-left-icon"`, `"link"` or `"link-left-icon"`. | `string`                            |
| suffixText   | A text to be displayed at the end of Input.                                                                                                                                                                  | `string`                            |
| prefixText   | A text to be displayed at the start of Input.                                                                                                                                                                | `string`                            |
| linkURL      | A URL for the link component.                                                                                                                                                                                | `string`                            |
| width        | To override or change the width of the component.                                                                                                                                                            | `string`                            |
| controlled   | If true the component is a controlled.                                                                                                                                                                       | `boolean`                           |
| onChange     | A Callback fired when the value is changed.                                                                                                                                                                  | `fn`                                |
| type         | Type of the input field                                                                                                                                                                                      | `number or text...`                 |
| required     | If true then a asterisk is displayed after the label(optional)                                                                                                                                               | `boolean`                           |
| id           | Id of the input field                                                                                                                                                                                        | `string`                            |
| name         | Name of the input field                                                                                                                                                                                      | `string`                            |
| ref          | Reference to the input field                                                                                                                                                                                 | `React.RefObject<HTMLInputElement>` |

`state`- Defines the state of the Input.

| value     | description                         |
| --------- | ----------------------------------- |
| default   | A normal input with basic features. |
| focused   | Makes the input focused.            |
| disabled  | A disabled input.                   |
| error     | To indicate an error.               |
| read-only | Acts as an label componenet.        |

`feature`- Indicates the kind of feature that user wants to consume.

| value                       | description                                          |
| --------------------------- | ---------------------------------------------------- |
| normal                      | A normal input with basic features.                  |
| left-icon                   | An input with Icon at the start.                     |
| right-icon                  | An input with Icon at the end.                       |
| left-right-icon             | An input with Icon at both start and end.            |
| indicative-text             | An input with text at end.                           |
| indicative-text-left-icon   | An input with text at end and Icon at the start.     |
| link                        | An input with Link at the end.                       |
| link-left-icon              | An input with Link at the end and Icon at the start. |
| prefix-text                 | An input with text at the start.                     |
| prefix-text-indicative-text | An input with text at both start and end.            |
| prefix-text-right-icon      | An input with text at start and icon at end.         |
