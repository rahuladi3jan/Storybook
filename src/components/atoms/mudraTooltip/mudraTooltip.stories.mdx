# Tooltip

Tooltips display informative text when users hover over, focus on, or tap an element.

## Usage

## 1. Tooltip Themes

You can change the theme of the tooltip by changing the `theme` attribute.

export const ThemeTooltip = () => {
  const lightRef = React.useRef(null);
  const darkRef = React.useRef(null);
  return (
    <CommonLayout>
      <div ref={lightRef}>
        <CommonComponent
          label={`Tooltip Themes`}
          direction={Direction.Bottom}
          beakSize={BeakSize.Medium}
          alignment={ToolTipAlignment.Start}
          gap={5}
          renderCondition={RenderCondition.Hover}
          theme={Theme.Light} // change the theme here
          parentRef={lightRef}
        />
      </div>
      <div ref={darkRef}>
        <CommonComponent
          label={`Dark Theme`}
          direction={Direction.Bottom}
          beakSize={BeakSize.Medium}
          alignment={ToolTipAlignment.Start}
          gap={5}
          renderCondition={RenderCondition.Hover}
          theme={Theme.Dark}
          parentRef={darkRef}
        />
      </div>
    </CommonLayout>
  );
};

## Props

<div style={{overflowX: 'auto'}}>

`MudraTooltip`

| prop            | description                                         | type                          |
| --------------- | --------------------------------------------------- | ----------------------------- |
| parentRef       | Reference for parent element                        | `ref`                         |
| direction       | Represents the placement of the tooltip             | `MudraTooltipDirection`       |
| alignment       | Alignment of the tooltip beak                       | `MudraTooltipAlignment`       |
| startAlignment  | The gap from start edge when the alignment is start | `number`                      |
| endAlignment    | The gap from end edge when the alignment is end     | `number`                      |
| beakSize        | Size of the beak                                    | `MudraTooltipBeakSize`        |
| isBeakHidden    | Controls the beak visibility                        | `boolean`                     |
| gap             | Gap between the tooltip and the component           | `number`                      |
| renderCondition | Represents the render condition of the tooltip      | `MudraTooltipRenderCondition` |
| theme           | Themes of the Tooltip                               | `MudraTooltipTheme`           |

</div>
