# Card

Card can be used as container for content to show.

## Usage

### 1. Basic Card

Card can used to show for some content of same type and format.

export const BasicCard = () => {
  return (
    <MudraCard>This is Mudra card component with a sample content.</MudraCard>
  );
};

<Canvas>
  <Story name="BasicCard">{BasicCard.bind()}</Story>
</Canvas>
