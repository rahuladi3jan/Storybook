# Button

Buttons allow users to take actions, and make choices, with a single tap.

## Usage

## 1. Button variants

Button with different variants can be consumed by changing the `variant` attribute and we have 3 different variant `Primary`, `Secondary` or `Ghost`

export const VariantButton = () => {
  return (
    <div
      style={{
        display: "flex",
        flexDirection: "row",
        justifyContent: "space-around",
      }}
    >
      <MudraButton variant={Variant.Primary} label="Primary Button" />
      <MudraButton variant={Variant.Secondary} label="Secondary Button" />
      <MudraButton variant={Variant.Ghost} label="Ghost Button" />
    </div>
  );
};

<Canvas>
  <Story name="Button Variants">{VariantButton.bind()}</Story>
</Canvas>

```jsx
import {
  MudraButton,
  MudraButtonVariant,
} from "@services/storybook-storybook/atoms/index";

const Component = () => {
  return (
    <MudraButton variant={MudraButtonVariant.Primary} label="Primary Button" />
  );
};
export default Component;
```

## 3. Button widths

Button with different width can be consumed by changing the `width` attribute and we have 4 different type `Auto`, `Full`, `Half` , `Circle` or `Square`

      />

```jsx
import {

  MudraButtonSize,

  MudraButtonIconAlignment,

```

## Props

<div style={{overflowX: 'auto'}}>

`MudraButton`

| prop          | description                       | type                     |
| ------------- | --------------------------------- | ------------------------ |
| variant       | The different variant             | MudraButtonVariant       |
| size          | The size of the component         | MudraButtonSize          |
| width         | The width of the component        | MudraButtonWidth         |
| iconAlignment | The alignment of the icon used    | MudraButtonIconAlignment |
| state         | Different states of the component | MudraButtonState         |
| label         | The text for the button           | `string`                 |
| icon          | Display Icon for component        | `SVG Icon`               |

</div>
