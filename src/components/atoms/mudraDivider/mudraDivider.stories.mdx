# Divider

Loader express an unspecified wait time or display the length of a process.

## Usage

```jsx
import { MudraDivider } from "./@services/storybook-storybook/atoms/mudraDivider";
```

### 1. Default Divider

MudraDivider comes with different styles `solid` and `dashed`.

<Canvas>
  <Story name="Basic Divider">
    <MudraDivider />
  </Story>
</Canvas>

```jsx
import { MudraDivider } from "./@services/storybook-storybook/atoms/mudraDivider";

const Component = () => {
  return <MudraDivider />;
};

export default Component;
```

### 2. Vertical Divider

## Props

<div style={{ overflowX: 'auto'}}>

`MudraDivider`

| prop     | description                                         | type      |
| -------- | --------------------------------------------------- | --------- |
| Vertical | A boolean value to indicate it's a vertical divider | `boolean` |
| bashed   | Divider of type dashed                              | `boolean` |
| color    | Specify the color of the divider                    | `string`  |

</div>
