# Badge

Badge can be used as label to show some small information or action.

## Usage

### 1. Basic Badge

Badge can be used to show some small information.

export const BasicBadge = () => {
  return (
    <MudraBadge
      label="Basic Badge"
      type={Type.Normal}
      theme={Theme.EmeraldGreen}
      size={Size.Medium}
    />
  );
};

<Canvas>
  <Story name="BasicBadge">{BasicBadge.bind()}</Story>
</Canvas>

```jsx
import {
  MudraBadge,
  Type,
  MudraBadgeSize,
  MudraBadgeTheme,
} from "@services/storybook-storybook/atoms/index";

export const BasicBadge = () => {
  return (
    <MudraBadge
      label="Basic Badge"
      type={Type.Normal}
      theme={MudraBadgeTheme.EmeraldGreen}
      size={MudraBadgeSize.Medium}
    />
  );
};
```

## Props

`MudraBadge`

<div style={{overflowX: 'auto'}}>

| prop               | description                                                     | type              |
| ------------------ | --------------------------------------------------------------- | ----------------- |
| `label`            | content of badge (Required)                                     | `string`          |
| `type`             | it defines what kind of badge it is. (optional)                 | `Type`            |
| `icon`             | icon user want to use                                           | `Svg`             |
| `theme`            | it defines the color user want to use.                          | `MudraBadgeTheme` |
| `size`             | it defines size of badge                                        | `MudraBadgeSize`  |
| `className`        | user can use this className for giving custom styles (optional) | `string`          |
| `onCloseIconClick` | function will be called on click of close icon                  | `function`        |

</div>

`type` : Type is an object with 4 keys

<div style={{overflowX: 'auto'}}>

| key         | description                       |
| ----------- | --------------------------------- |
| `Normal`    | It has a label only.              |
| `WithDot`   | It has a dot with label.          |
| `WithClose` | It has close icon with label.     |
| `IconLeft`  | It has icon on the left of label. |

</div>

`size` : MudraBadgeSize is an object with 3 keys

<div style={{overflowX: 'auto'}}>

| key      | description               |
| -------- | ------------------------- |
| `Small`  | Badge has small in size.  |
| `Medium` | Badge has medium in size. |
| `Large`  | Badge has large in size.  |

</div>
