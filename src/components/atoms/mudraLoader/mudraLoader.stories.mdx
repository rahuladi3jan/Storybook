# Loader

Loader express an unspecified wait time or display the length of a process.

## Usage

```jsx
import { MudraLoader } from "./@services/storybook-storybook/atoms/mudraLoader";
```

### 1. Default Loader

MudraLoader comes with different sizes `small`, `medium (default)`, `large`, `xl` and `xxl`.

<Canvas>
  <Story name="Basic Loader">{LoaderTemplate.bind()}</Story>
</Canvas>

```jsx
import {
  MudraLoader,
  MudraLoaderSize,
} from "./@services/storybook-storybook/atoms/mudraLoader";

const Component = () => {
  return <MudraLoader size={MudraLoaderSize.small} />;
};
export default Component;
```
