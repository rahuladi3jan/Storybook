# Autocomplete

Autocomplete components is a similar component like a basic select with the option to search and select options.

```jsx
import { MudraAutocomplete } from "./@services/storybook-storybook/molecules/mudraAutocomplete";

const Page = () => {
  return (
    <MudraAutocomplete
      placeHolder="Basic Autocomplete"
      options={[
        { value: "green", label: "Green" },
        { value: "blue", label: "Blue" },
        { value: "red", label: "Red" },
        { value: "yellow", label: "Yellow" },
        { value: "orange", label: "Orange" },
        { value: "pink", label: "Pink" },
        { value: "purple", label: "Purple" },
        { value: "grey", label: "Grey" },
      ]}
      onSelect={(value) => {
        console.log("selected", value);
      }}
    />
  );
};

export default Page;
```

## Props

<div style={{overflowX: 'auto'}}>

| prop          | description                                                           | type                                        |
| ------------- | --------------------------------------------------------------------- | ------------------------------------------- |
| `placeHolder` | The short hint displayed in the select before the user select a value | `string`                                    |
| `options`     | Array of object with key value and label used for listing options     | `[{ value: string/number, label: string }]` |
| `onSelect`    | function(value: object) => void                                       | `func`                                      |
| `value`       | Object thet represents selected option                                | `{ value: string/number, label: string }`   |
| `className`   | Custom class name                                                     | `string`                                    |

</div>
