# TextArea

- The textarea tag defines a multi-line text input control.
- The textarea element is often used in a form, to collect user inputs like comments or reviews.

## Usage

## 1. Disabled TextArea

TextArea with disabled feature can be enabled by making the `disabled` attribute `true`

<Canvas>
  <Story
    name="Disabled TextArea"
    args={{
      defaultValue: "test",
      rows: 2,
      label: "Label - Default",
      hintText: "hintText",
      hint: true,
      disabled: true,
    }}
  >
    {TextAreaTemplate.bind()}
  </Story>
</Canvas>

```jsx
import { MudraTextArea } from "./@services/storybook-storybook/molecules/mudraTextArea";

const Component = () => {
  return (
    <MudraSelect
      defaultValue="test"
      rows={2}
      label="Label - Default"
      hintText="hintText"
      hint={true}
      disabled={true}
    />
  );
};

export default Component;
```

## Props

`MudraTextArea`

<div style={{overflowX: 'auto'}}>

| prop        | description                               | type      |
| ----------- | ----------------------------------------- | --------- |
| error       | An error indication for component         | `boolean` |
| disabled    | A attribute to disable the component      | `boolean` |
| required    | value to be true for the required field   | `boolean` |
| placeholder | placeholder for the text area             | `string`  |
| label       | A text to display the label for component | `string`  |
| hint        | A control to display the hintText         | `boolean` |
| hintText    | A text to display the hint for component  | `string`  |

</div>
