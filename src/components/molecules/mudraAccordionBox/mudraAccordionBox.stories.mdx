# Accordion Box

Accordion Box can be used to wrap one or more Accordions.
`id` prop should always be different for all the AccordionBox you are using in your project.

`The content of Accordion Box is mounted by default even if the accordion is not expanded. This default behavior supports server-side rendering and SEO.`

## Usage

### 1. Basic Accordion Box

It will open and close on click. At one time multiple number of accordions can be expanded and by default all will be closed.

export const BasicAccordionBox = () => {
  return (
    <div style={{ margin: "16px" }}>
      <MudraAccordionBox id="boxidOne">
        <MudraAccordion id="acc_one" title="Accordion One">

            <p>Accordion Content</p>

## Props

`MudraAccordionBox`

<div style={{overflowX: 'auto'}}>

| prop                           | description                                                                                                                 | type                    |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------------------- | ----------------------- |
| `title`                        | Title of accordion box (Optional)                                                                                           | `string`                |
| `className`                    | user can use this className for giving custom styles (optional)                                                             | `string`                |
| `type`                         | `Basic` or `Advance` - decides how if one or more accordions can be expanded. (default Basic)                               | `MudraAccordionBoxType` |
| `defaultExpanded`              | decide which accordion will be default expanded when component will render. (optional)                                      | `string`                |
| `children`                     | `Accordion`                                                                                                                 | `Accordion`             |
| `id`                           | id of accordion box to differentiate it from other accordion box used in project. (required and should always be unique)    | `string`                |
| `intiallyAllAccordionExpanded` | `true` it will make all accordion open initially and will work with only basic accordion. (Optional, default will be false) | `boolean`               |

</div>

`MudraAccordionBoxType` : It has two keys `Basic` and `Advance` by default it will be `Basic`.

<div style={{overflowX: 'auto'}}>

| prop      | description                                           |
| --------- | ----------------------------------------------------- |
| `Basic`   | Multiple accordions can be kept expanded at one time. |
| `Advance` | Only one accordion can be expanded at one time.       |

</div>
