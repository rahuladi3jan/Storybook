# Accordion

Accordion molecule allows the user to show and hide sections of related content on a page.
An accordion is a lightweight container that may either be used standalone, or be connected to a larger surface, such as a card.

`The content of Accordions is mounted by default even if the accordion is not expanded. This default behavior supports server-side rendering and SEO.`

## Usage

### 1. Basic Accordion

It will open and close on click, user won't have to pass any function.
User need to pass title as string.
The content user want to show on accordion open should be passed as children.
User can pass `initiallyExpanded` as `true` or `false` for initial condition for accordion.

export const BasicAccordion = () => {
  return (
    <div style={{ margin: "16px" }}>
      <MudraAccordion id="acc_one" title="Contact Details">
        <div
          style={{ display: "flex", flexDirection: "row", marginTop: "12px" }}
        >
          <div>
            <h1 style={{ fontWeight: "600" }}>Mobile Number</h1>
            <p style={{ marginTop: "8px" }}>0000000000</p>
          </div>
          <div style={{ marginLeft: "48px" }}>
            <h1 style={{ fontWeight: "600" }}>Personal Email ID</h1>
            <p style={{ marginTop: "8px" }}>abcxyz@gmail.com</p>
          </div>
        </div>
      </MudraAccordion>
    </div>
  );
};

## Props

`MudraAccordion`

<div style={{overflowX: 'auto'}}>

| prop                | description                                                                                                | type                     |
| ------------------- | ---------------------------------------------------------------------------------------------------------- | ------------------------ |
| `title`             | Title of accordion (Required)                                                                              | `string`                 |
| `className`         | user can use this className for giving custom styles (optional)                                            | `string`                 |
| `initiallyExpanded` | `true` or `false` - this decides the open and close state of accordion initially (optional, default false) | `boolean`                |
| `children`          | html content user want to show when accordion is open.                                                     | `ReactElement`           |
| `icon`              | icon display at left side of accrodion header (Optional)                                                   | `string`                 |
| `iconAlignment`     | The alignment of the icon used                                                                             | `IconAlignment`          |
| `onChange`          | Callback triggered when the Accordion collapsed or expanded                                                | `(AccordionState)=>void` |

</div>
